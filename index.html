MultInput

MultInput is a simple jQuery plugin that splits your form inputs into multiple inputs using a pattern.

Download multinput.js

Features

Lightweight jQuery plugin.

JavaScript and CoffeeScript versions.

CSS and SASS styling.

Available markup API via data attributes.

Getting started

Just make sure to include the following scripts on your HTML page:

<script src="jquery.js"></script>
<script src="jquery.multinput.js"></script>

Add a pattern to the input you want to split:

<input type="text" data-multinput data-pattern="ES-99-9999-9999-9999-9999-9999">

Options

Data attributes

  • data-multipart: Add this attribute to automatically apply the plugin to the inputs you desire.
  • data-pattern: The pattern which will be create the inputs.
  • data-separator: The char used with the help of the pattern to split the input.
  • data-classes: Additional classes you can add to the inputs created to apply your own styles.
